# Budget-Management-Backend
An application for managing the home budget and property rental


## TECH STACK:
- Spring Boot 2.6.4 + Java 17 + Maven
- Spring Security + JWT
- Spring Data JPA
- MySQL Db + H2 Db
- FlywayDb
- Lombok
- Swagger
- Mapstruct
- JUnit 5
- Mockito
- AssertJ
- Design patterns:
{ Builder + Abstract Factory + Chain of Responsibilities + Strategy }
